## Описание программы

*Currency Converter — это GUI‑приложение для конвертации валют с использованием внешнего API. Позволяет выбирать валюты для конвертации, вводить сумму, получать актуальные курсы и просматривать историю операций в табличном виде.*

---

## Требования для запуска

Что нужно установить на компьютер:

* Python 3.10 или выше;
* Библиотеки: `pip install requests`.

---

## Как запустить

```bash
git clone https://github.com/твой-логин/currency-converter.git
cd currency-converter
pip install -r requirements.txt
python main.py

## Получение API‑ключа
Перейдите на сайт exchangerate-api.com.

Зарегистрируйтесь и создайте аккаунт.

Получите бесплатный API‑ключ в личном кабинете.

Замените значение YOUR_API_KEY в коде (main.py) на ваш ключ:
self.api_key = "ваш_api_ключ"

## Особенности реализации
GUI: построен на tkinter (встроенная библиотека Python).

API: используется exchangerate-api.com для получения актуальных курсов.

История: сохраняется в формате JSON, отображаются последние 10 записей.

Валидация: проверка корректности ввода суммы и выбора валют.

Обработка ошибок: уведомления для пользователя при сетевых ошибках или некорректном вводе.
