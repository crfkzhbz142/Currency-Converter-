## Описание программы

*Currency Converter — это GUI‑приложение для конвертации валют с использованием внешнего API. Позволяет выбирать валюты для конвертации, вводить сумму, получать актуальные курсы и просматривать историю операций в табличном виде.*

## Требования для запуска

Что нужно установить на компьютер:

* Python 3.10 или выше;
* Библиотеки: `pip install requests`.

## Как запустить

git clone https://github.com/твой-логин/currency-converter.git
cd currency-converter
pip install -r requirements.txt

## Как получить API‑ключ

Для работы приложения необходим API‑ключ от сервиса обмена валют exchangerate-api.com. Пошаговая инструкция:

1. Откройте браузер и перейдите на сайт: https://www.exchangerate-api.com
2. Нажмите кнопку **Sign Up** (Зарегистрироваться) в верхнем меню.
3. Заполните форму регистрации:
   * укажите email;
   * придумайте пароль;
   * примите условия использования.
4. Подтвердите email (проверьте почтовый ящик, перейдите по ссылке в письме).
5. Войдите в личный кабинет (Login).
6. В панели управления найдите раздел **API Keys** или **Dashboard**.
7. Создайте новый API‑ключ (кнопка **Create API Key** или аналогичная).
8. Скопируйте сгенерированный ключ — это строка вида `YOUR_UNIQUE_API_KEY_123`.
9. Откройте файл `main.py` в текстовом редакторе.
10. Найдите строку:
    ```python
    self.api_key = "YOUR_API_KEY"
    ```
11. Замените `"YOUR_API_KEY"` на ваш реальный ключ, например:
    ```python
    self.api_key = "abc123def456ghi789"
    ```
12. Сохраните изменения в файле `main.py`.

> **Важно:** не публикуйте свой API‑ключ в открытом доступе. Если ключ скомпрометирован, сгенерируйте новый в личном кабинете.
