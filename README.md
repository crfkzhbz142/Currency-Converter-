## Описание программы

*Currency Converter — это GUI‑приложение для конвертации валют с использованием внешнего API. Позволяет выбирать валюты для конвертации, вводить сумму, получать актуальные курсы и просматривать историю операций в табличном виде.*

---

## Требования для запуска

Что нужно установить на компьютер:

* Python 3.10 или выше;
* Библиотеки: `pip install requests`.

---

## Как запустить

```bash
git clone https://github.com/твой-логин/currency-converter.git
cd currency-converter
pip install -r requirements.txt
python main.py
